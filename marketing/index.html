FAQ

常见问题

需要编程知识吗？

不需要。AI Agent 自动完成所有技术工作。你只需要在仪表盘上查看结果和做决策。

与 Polsia 原版有什么区别？

这是 Polsia 的开源分支版本。将底层从 Claude Code CLI 替换为标准 LLM API（兼容 DeepSeek、OpenAI 等），降低了使用门槛和成本。

需要什么硬件？

开源版可以运行在任何 VPS 或云服务器上（2GB 内存即可），也支持 Docker 一键部署。

AI 会泄露我的商业信息吗？

开源版完全自托管，数据不出你的服务器。托管版使用企业级加密存储。

支持中文吗？

支持。前端已完整中文化，Agent 可以用中文输出内容、生成报告。

如何开始？

克隆 GitHub 仓库，运行 docker-compose up，5 分钟启动你的 AI 公司。

准备好让 AI 帮你运营公司了吗？

不需要团队，不需要技术背景。10 个 AI Agent 每天为你完成所有工作。
