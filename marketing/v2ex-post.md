## V2EX 发帖草案

### 标题
一人公司福音：我 Fork 了 Polsia（$1000万ARR），开源全栈 AI 自动运营系统

### 正文

先说说背景。我是一个人的开发者，去年开始研究 Polsia——这家公司估值 $2.5 亿、单人运营、ARR 接近 $1000 万，号称「AI 自动运行公司」。研究完我发现：**他们的核心代码根本不在开源仓库里**，开源的部分只是一个空壳。

于是我从零重写了整个后端（Python），保留了前端（Next.js），造了一个可用的 Polsia Fork。

### 能做什么

10 个 AI Agent 每天早上自动工作：

| Agent | 干什么 |
|-------|--------|
| CEO/Orchestrator | 规划每日任务、分派工作、生成日报 |
| 商业分析 | 市场研究、战略建议 |
| 竞品监控 | 自动追踪对手动态 |
| 社媒运营 | 内容创作、定时发帖 |
| 邮件营销 | 客户开发、邮件自动化 |
| 广告管理 | 投放优化、预算追踪 |
| 客服 | 工单分类、自动回复 |
| 代码生成 | 开发任务自动化 |
| 财务 | 收入/支出追踪 |
| 部署 | 基础设施管理 |

### 技术栈
- **后端**: Python 3.12 + FastAPI + Celery + PostgreSQL
- **前端**: Next.js 14（中英文双语）
- **LLM**: DeepSeek API（兼容 OpenAI 格式），自带成本追踪
- **测试**: 74 个测试全部通过

https://github.com/guish7423/polsia-fork
